# Planung-Tool Issues

User-Wünsche „für später", die nicht in der laufenden Session umgesetzt werden.
Beim Session-Start auflisten und den User fragen, woran weitergearbeitet werden soll.

Status-Konvention:
- `OPEN` — noch nicht begonnen
- `IN_PROGRESS` — angefangen
- `DONE` — erledigt, kann archiviert/entfernt werden
- `BLOCKED` — wartet auf Input vom User (z.B. Key, Liste)
